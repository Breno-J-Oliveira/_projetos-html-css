## O que é esta categoria?

A categoria **Projetos HTML/CSS** é o coração visual do repositório — **12 projectos** que mostram a evolução do design web, desde a primeira página estática até um jogo completo de Tetris com Canvas API 2D. São sites institucionais, landing pages, campanhas temáticas, protótipos no Figma e aplicações interativas com JavaScript.

É aqui que a criatividade encontra o código: cada projecto tem uma identidade visual única, seja um site romântico com corações flutuantes e som de batimento cardíaco, uma campanha de conscientização azul, ou um jogo clássico implementado em 258 linhas de JavaScript puro.

**Do primeiro `<h1>` ao game loop do Tetris.**

## Destaque: Jogo Tetris (Projecto 12)

O **Tetris** é o projecto tecnicamente mais impressionante desta categoria. Implementação completa do clássico jogo em **258 linhas** de JavaScript vanilla, renderizado inteiramente via **Canvas API 2D**.

### Mecânica do Jogo

| Componente | Especificação |
|------------|---------------|
| **Grid** | 10 colunas × 20 linhas (30px cada) |
| **Canvas** | 300px × 600px |
| **Peças** | 7 tetrominós clássicos (I, O, T, S, Z, J, L) |
| **Cores** | Cada peça tem cor distinta: `#00FFFF`, `#FFFF00`, `#800080`, `#00FF00`, `#FF0000`, `#0000FF`, `#FFA500` |

### Sistema de Pontuação

| Linhas | Pontos | Nome |
|--------|--------|------|
| 1 | 10 | Single |
| 2 | 20 | Double |
| 3 | 30 | Triple |
| 4 | 40 | **Tetris!** |

### Controles

| Tecla | Ação |
|-------|------|
| ← → | Mover peça |
| ↓ | Queda rápida (soft drop) |
| ↑ / Q / W | Rotação |
| Espaço | Queda instantânea (hard drop) |

### Progressão de Nível
- A cada 100 pontos, o nível aumenta
- Cada nível reduz o tempo de queda em 50ms
- Slider de velocidade ajustável (100ms a 1000ms)

## Como Explorar Esta Categoria

### Pré-requisitos
- Navegador web (Chrome, Firefox, Edge)
- Editor de código (VS Code recomendado)
- Leitor de PDF (para o protótipo Spotify e documentação ABNT)
- Figma (opcional, para abrir o protótipo)

### Passo a passo

1. **Começa pelo Matrix Revolutions** — abre `index.html` no navegador, vê como se estrutura uma página simples
2. **Explora o Desafio SENAI** — redimensiona o navegador para ver a responsividade em ação
3. **Abre o Dia dos Namorados** — vê a integração de vídeo, áudio e animações JS
4. **Joga o Tetris** — abre `tetris2/index.html`, testa os controles, ajusta a velocidade no slider
5. **Analisa o código do Tetris** — estuda o game loop, a lógica de colisão, a renderização Canvas
6. **Vê o protótipo Spotify** — abre o PDF e observa o design das telas de login e home

### Estrutura de cada projecto

Cada subpasta contém:
- **`README.md`** — objectivos, conceitos, tecnologias
- **`index.html`** — página principal
- **`style.css`** (quando separado) — estilos
- **Assets** — imagens, fontes, vídeos, áudios, protótipos
